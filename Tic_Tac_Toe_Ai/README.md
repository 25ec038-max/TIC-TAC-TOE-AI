TIC-TAC-TOE AI
Project Description
Project Title: Tic-Tac-Toe AI Using Minimax Algorithm

Abstract
The Tic-Tac-Toe AI project is a Python-based implementation of the classic Tic-Tac-Toe game where a human player competes against an Artificial Intelligence (AI). The AI uses the Minimax algorithm, a decision-making algorithm in Artificial Intelligence, to determine the best possible move. Since the algorithm evaluates every possible game state, the AI always plays optimally, making it impossible to defeat.

The project demonstrates the practical application of AI concepts such as game trees, recursion, adversarial search, and optimal decision-making. It is ideal for beginners learning Artificial Intelligence and Python programming.

Objectives
Develop a console-based Tic-Tac-Toe game.

Implement Artificial Intelligence using the Minimax Algorithm.

Allow a human player to play against the computer.

Demonstrate optimal decision-making in games.

Learn recursion and game-tree search techniques.

Features
Human vs AI gameplay

AI never loses

Minimax Algorithm implementation

User-friendly interface

Detects winner and draw

Input validation

Clean Python code

Easy to understand

Algorithm
Start the game.

Create an empty 3×3 board.

Human enters a move.

Validate the move.

Update the board.

Check for winner or draw.

AI calculates the best move using Minimax.

AI places its move.

Repeat until game ends.

Display winner or draw.
